{% comment %} EXPLANATION FOR JUNIOR DEVELOPERS: ==================================== 1. {{ greeting }} - This displays the 'greeting' variable that we passed from the view 2. {{ app_name }} - This shows the 'app_name' from our context dictionary 3. {{ current_time|date:"..." }} - This applies a date filter to format the timestamp 4. {% url 'data_types' %} - This generates a URL to another view using the URL name The data flows like this: VIEW (Python) → CONTEXT (Dictionary) → TEMPLATE (HTML) → USER (Browser) In our view, we had: context = { 'greeting': 'Hello, World!', 'current_time': datetime.now(), 'app_name': 'My Django App', 'version': '1.0.0' } And here in the template, we access these values using {{ variable_name }} {% endcomment %} {% endblock %}
